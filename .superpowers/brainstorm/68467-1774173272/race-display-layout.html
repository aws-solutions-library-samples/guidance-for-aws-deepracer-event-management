Race display layout — 53×11 Galactic Unicorn

53 pixels wide × 11 tall is tight. Which layout works best for the race state?

← scrolling text cycles through all data →

A — Single scrolling line

One line of large-ish text scrolls continuously: ⏱ 02:34 · DAVE · 3 laps · best 12.45s · last 13.21s. Simple, readable from a distance. Reset events flash the display red briefly.

top: time remaining · bottom: scrolling racer/laps/best

B — Two rows

Top row (5px tall): time remaining, always visible in yellow. Bottom row (5px tall): scrolls through racer name → laps → best lap → last lap. Reset flashes bottom row red.

left: laps + resets (static) · right: scrolling time info

C — Split: pinned + scrolling

Left ~12px: lap count and reset count always visible (static). Right ~38px: scrolls through racer name, time remaining, best lap, last lap. Reset counter increments live and flashes orange.
